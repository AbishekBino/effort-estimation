About This Project
Machine Learning based Software Effort & Cost Estimation System
🎯
Problem Statement
Manual software effort estimation leads to project delays and budget overruns. This ML system automates estimation using historical project data.
🗂️
Datasets
Desharnais dataset (81 projects, 10 features) and Combined dataset (642 projects, 3 features) — both real-world software engineering data.
🤖
Best Model — MLP
Multi-Layer Perceptron with architecture 3→32→16→1. ReLU activation, Adam optimizer. R²=0.646, consistent with IEEE published research.
📊
R² = 0.646 — Is it good?
IEEE and ACM papers on the same Desharnais dataset report R² of 0.55–0.72. Our result is within published research range.
🌐
Deployment
Hosted on Streamlit Cloud via GitHub. Live 24/7, accessible from any device, any network — no local setup required.
🔮
Future Work
SHAP explainability, more features (team size, complexity, risk), Docker containerization, CI/CD pipeline integration.
Technology Stack
Python 3.13 Scikit-learn Streamlit Pandas NumPy Matplotlib Joblib GitHub Streamlit Cloud
